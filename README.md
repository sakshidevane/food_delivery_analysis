
# Food Delivery Data Analysis Project

This project analyzes food delivery data using Python and Jupyter Notebook.

## Dataset Used
- orders.csv (order transactions)
- users.json (user details)
- restaurants.sql (restaurant data)

## Tools & Technologies
- Python
- Pandas
- SQLite
- Matplotlib
- Jupyter Notebook

## Project Tasks
- Data loading from CSV, JSON, SQL
- Data merging using LEFT JOIN
- Data cleaning and preprocessing
- Exploratory data analysis
- Insights generation

## Output
- final_food_delivery_dataset.csv
